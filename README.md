# Vascular_Param_Preds

This work aims to apply various methods in 3D vascular models to predict time-series hemodynamic parameters.

This work follows a four-part investigation:

1) A classification DGCNN model has been used and to find the optimal parameters that minimize loss and allow for the best prediction accuracy;
2) The impact of skeleton inclusion on prediction accuracy has been studied;
3) The application of data augmentation has been evaluated to improve our predictions;
4) A segmentation DGCNN model has been used and prediction accuracy has been compared. The use of the PINN model has been then analyzed as a complement to the segmentation DGCNN model.
