# Vascular_Param_Preds

This work aims to apply various methods to 3D vascular models in order to predict time-series hemodynamic parameters.

The study is structured into four main parts:

1- A classification DGCNN model is used to identify the optimal parameters that minimize loss and maximize prediction accuracy;
2- The impact of skeleton inclusion on prediction accuracy is investigated;
3- The effectiveness of data augmentation in improving predictions is evaluated;
4- A segmentation DGCNN model is implemented, and its prediction accuracy is compared with previous approaches. Additionally, the use of a PINN model is analyzed as a complement to the segmentation DGCNN model.
